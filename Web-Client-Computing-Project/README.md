# Find Your Wave 🌊

> 2026학년도 1학기 웹클라이언트컴퓨팅 02분반 팀 프로젝트

서핑의 정신을 웹으로 옮긴 인터랙티브 웹사이트. 파도를 테마로 한 게임, 성향 테스트, 여행 체크리스트, 팀원 소개 페이지로 구성되어 있습니다.

---

## 📖 프로젝트 소개

**"파도는 기다리는 자에게 온다"**

서핑은 단순한 스포츠가 아니라 바다와 하나가 되는 순간, 자연의 리듬에 몸을 맡기고 나만의 파도를 찾아 나서는 여정입니다. 이 프로젝트는 그런 서핑의 철학을 다섯 개의 페이지로 풀어낸 웹 애플리케이션입니다.

## ✨ 주요 기능

### 🎮 Surf Game — Wave Rider
- HTML5 Canvas로 구현된 종스크롤 장애물(섬) 피하기 게임
- 장애물에 맞아 목숨 3개가 사라지면 게임 오버
- 캐릭터에서 총알 나오는 것 삭제하기
- 보스몬스터 등 각종 몬스터 등장하는 것 삭제
- 키보드(WASD / 방향키) 및 모바일 터치 드래그 조작 지원
- 게임 화면 밑에 잠금 되어있는 총 7종류의 해변 도감이 있음. 스탬프로 해금 가능
- 스탬프 획득 기능: 지역별 서핑 스팟 및 해변의 사진과 정보를 해금할 수 있는 7종류의 스탬프를 랜덤으로 획득할 수 있음. 
- 스탬프 디자인: 빨간 원형 스탬프 모양에 분별하기 좋게 stamp 글자가 적혀있음.


### 🌊 Find My Wave — 성향 테스트
- 이름, 라디오, 셀렉트, 텍스트에어리어 등 다양한 폼 요소 활용
- 클라이언트 사이드 유효성 검증
- 답변 조합에 따른 개인화된 서핑 스타일 결과 제공

### 🧳 Travel Checklist
- 4가지 테마 선택 (서핑 초보자 / 해변 힐링 / 서핑 고수 / 비치 파티)
- 테마별 기본 준비물 자동 생성
- 사용자 정의 항목 추가 가능
- 진행률 표시 및 완료 항목 일괄 삭제 기능

### 🎨 공통 디자인
- 반응형 레이아웃 (모바일 햄버거 메뉴 포함)
- 파도 SVG 애니메이션
- 일관된 컬러 팔레트 (오션 블루 + 민트)

## 🚀 실행 방법

별도의 빌드 과정이 필요 없는 정적 웹사이트입니다.

### 방법 1. 브라우저에서 바로 열기
```
team_project/index.html 파일을 더블클릭하여 브라우저에서 실행
```

### 방법 2. 로컬 서버 사용 (권장)
```bash
# Python 3
cd team_project
python3 -m http.server 8000

# Node.js (http-server 설치 시)
npx http-server team_project -p 8000
```
브라우저에서 `http://localhost:8000` 접속

### 방법 3. VS Code Live Server 확장
`index.html` 우클릭 → **Open with Live Server**
